# CDDA Game Launcher

A Cataclysm: Dark Days Ahead launcher with additional features.

## Planned features

* Basic game installer
* Automatic updates and version detection
* Mod manager
* Tileset manager
* Soundpack manager
* Font presets
* Regular save backups
